清除浮动的方式

语法： 选择器{clear:both;}

清除方式

  • 额外标签法(不常使用)
    在浮动元素末尾添加一个空标签：
  • 父级元素添加overflow属性
    给父级元素添加overflow:hidden/auto/scroll;三种都可实现。
  • after伪元素清除浮动
    语法：
    .clearfix:after{ content:".",
                 display:block,
                 height:0,
                 visibility:hidden,
                 clear:both;
                }
.clearfix { *zoom:1;}  ie6、7专有
  • 空余字符法
    语法：
    .clearfix:after{ content:"\200B",
                 display:block,
                 height:0,
                 clear:both;
                }
.clearfix { *zoom:1;}  ie6、7专有
  • 双伪元素法
    语法：
    .clearfix:after,.clearfix:after{ content:" ",
                                 display:table,
                                }
.clearfix:after{ clear:both;}
.clearfix { *zoom:1;}  ie6、7专有
